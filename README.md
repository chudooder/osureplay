# osureplay
An osu! replay parsing system designed for the web.

# Setup

```bash
git clone git@github.com:chudooder/osureplay.git

mkdir oppai
cd oppai
git clone https://github.com/Francesco149/oppai.git

cd ..
npm install
cd public
bower install
```

To run the server, in the home directory:
```bash
node server.js
```
